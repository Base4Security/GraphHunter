************************
Privacy & data
************************

.. contents:: Table of Contents

Graph Hunter is designed so that **all processing stays on your machine**.

* **Logs** — Read only from files you explicitly select. No log data is sent to external services.
* **Sessions** — Stored locally in your OS application data directory (e.g. ``%APPDATA%`` on Windows, ``~/Library/Application Support`` on macOS, ``~/.local/share`` on Linux).
* **Notes and path nodes** — Saved with the session in the same local storage.
* **Telemetry** — No analytics, usage tracking, or phone-home functionality is included.

You can use Graph Hunter on sensitive or air-gapped environments without exposing data to the internet.
